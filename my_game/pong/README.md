# Pong (Love2D)

Simple two-player Pong built for LÃ–VE (Love2D).

How to run
- From the workspace root run:
```
love my_game/pong
```

Controls
- Left paddle: `W` (up), `S` (down)
- Right paddle: `Up` and `Down` arrow keys
- `Space` to pause/unpause
- `R` to reset scores
- `Esc` to quit

Single-player
- By default the right paddle is controlled by AI. Press `Tab` to toggle between Single Player (AI) and Two Player modes.
 
Difficulty Menu
- Press `M` to open the Difficulty menu. Use the slider with mouse drag or use the left/right arrow keys to adjust AI skill (0.0 = easiest, 1.0 = hardest).
- The current AI skill is shown on-screen when the menu is open.


Notes
- The game is a minimal implementation intended for learning and extension.
- To play inside Codespaces with GUI forwarding, ensure your display is available.
